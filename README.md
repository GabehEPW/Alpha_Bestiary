🧪 Bestiary Alpha

Meu primeiro mod sendo desenvolvido para o Terraria, com foco em progressão dinâmica e incentivo à exploração completa do jogo.

📌 Sobre o Mod

Este mod introduz um sistema inovador de progressão de armas baseado na diversidade de inimigos derrotados.
Ao invés de evoluir apenas por quantidade de abates ou farming repetitivo, o jogador evolui seus equipamentos ao enfrentar diferentes tipos de inimigos ao longo da jornada.

⚔️ Como Funciona

Cada arma é vinculada a uma build específica (Ranged, Melee, Magic, etc).
O nível da arma aumenta ao derrotar um novo tipo de inimigo.
Derrotar o mesmo inimigo novamente não aumenta o nível.

🌎 O objetivo é incentivar o jogador a:

- Explorar todos os biomas e suas respectivas cavernas
- Enfrentar diferentes inimigos 
- Participar de eventos variados
